## 🚀 使用指南

### 启动项目

1. **启动后端**
```bash
cd api
python -m uvicorn main:app --host 0.0.0.0 --port 9000 --reload
```

2. **启动前端**
```bash
cd web
npm install
npm run dev
```

3. **访问应用**
- 前端：http://localhost:5173
- API 文档：http://localhost:9000/docs

### 配置说明

各 Engine 的配置需要在对应的 `config.py` 文件中设置：
- LLM API Key 和 Base URL
- Tavily API Key（用于搜索）
- 数据库连接信息（如使用）

## 📝 技术栈

### 前端
- **框架**：React 19 + TypeScript
- **构建工具**：Vite 7
- **样式**：TailwindCSS 4.0
- **Markdown**：React Markdown + remark-gfm
- **状态管理**：React Hooks

### 后端
- **框架**：FastAPI
- **异步**：asyncio
- **数据验证**：Pydantic
- **流式输出**：Server-Sent Events
